# 考试系统

#### Description
系统基于spring boot + vue开发，前端框架elementui，在若依/Ruoyi（Vue前后端分离版本）整体系统框架上开发了考试系统相关。
整个系统包括试题管理、考试组织、网上考试、以及资料管理四部分，资料管理只是作为图片、文件共享使用，跟考试系统无必然联系，考试系统只包括试题管理、考试组织、网上考试等。
